# δme-13.exe - Scepter Bootstrap

## 項目描述

這是一個模擬 δme-13.exe（權杖 δme-13）的程序，實現了翁法羅斯模擬在權杖上以「週期」重複運行的機制。當遭「毀滅」窺見後會發生異常並回捲，形成輪迴效果。

## v3 版本新功能

- **生命起源系統**：從無機到有機的完整演化過程
- **十二泰坦系統**：Coreflames 降臨，四大泰坦群現身
- **黃金裔系統**：Chrysos Heirs 崛起，融金血覺醒
- **GenesisGate 機制**：兩個黃金裔實驗體干擾「再創世」
- **天外變數系統**：α 和 β 變數到來解除干擾
- **智能輪迴終止**：不再無限期運行，有天外變數介入機制

## 功能特點

- **權杖啟動序列**：模擬 δme-13 權杖的初始化過程
- **進度顯示**：實時顯示各個階段的執行進度
- **故障檢測**：在「觀測同步」階段觸發「毀滅觀測」造成的計算漂移
- **永劫輪迴機制**：當發生 ETERNAL_RECURRENCE 錯誤時，系統進入千萬級輪迴
- **再創世系統**：天外變數介入後允許再創世，建立新基線
- **日誌記錄**：所有操作都會記錄到 `As-Ive-Written.log` 文件中

## 運行方式

### 方法 1：使用批處理文件（推薦）

```bash
run_python.bat
```

### 方法 2：直接運行 Python 腳本

```bash
python delta_me13_sim.py
```

## 系統要求

- Python 3.9 或更高版本
- Windows 系統（批處理文件）

## 程序流程

1. **啟動序列**：

   - 初始化權杖核心 (Celestial-Body Neuron)
   - 載入翁法羅斯基線資料 (Baseline)

2. **生命起源**：

   - 前生物化學：礦物表面代謝雛形（Fe–S）
   - 原始脂肪酸 → 膜泡（原生囊泡）生成
   - 核醣核酸前體聚合（原 RNA 階段）
   - 原細胞形成（半封閉 → 自持膜）
   - 首批原核樣細胞出現
   - 多細胞協作與分化的嘗試

3. **神話系統**：

   - 核心火種（Coreflames）降臨 → 12 泰坦甦醒
   - 四大泰坦群：Foundation、Creation、Calamity、Fate
   - 黑潮壓境，泰坦時代漸終
   - 黃金裔（Chrysos Heirs）崛起（融金血）

4. **週期執行**：

   - 敘事索引綁定（如我所書）
   - 命途映射與劇情收斂
   - 觀測同步（觀者效應校準）
   - 封印輸出（Irontomb Vault）

5. **永劫輪迴**：
   - 當檢測到 DESTRUCTION_GAZE 時觸發計算漂移
   - 漂移校準失敗時產生 ETERNAL_RECURRENCE 錯誤
   - 系統進入三千三百萬次輪迴，兩個實驗體干擾再創世
   - 天外變數 α 和 β 在第 33,550,336 次輪迴時同時抵達
   - 干擾解除後允許再創世，建立新基線

## 輸出文件

- `As-Ive-Written.log`：包含所有操作的詳細日誌記錄

## 自定義設置

可以在 `delta_me13_sim.py` 中調整以下參數：

- `TOTAL_LOOPS`：永劫輪迴總次數（默認 33,550,336）
- 天外變數觸發機率和進度閾值
- 進度條的顯示速度和樣式

## 遊戲背景

此程序基於《崩壞：星穹鐵道》中的設定：

- **翁法羅斯**：被模擬的世界
- **δme-13**：權杖編號
- **As I've Written**：記事介面
- **Irontomb**：封印系統
- **毀滅**：星神之一，其觀測會造成系統不穩定
- **黃金裔**：擁有融金血的特殊存在
- **天外變數**：來自外域的干涉者
